## Run Locally

### Prerequisites

- Node.js (v14+ recommended)
- npm or yarn

### Steps

Clone the project

```bash
  git clone https://github.com/Samyak0-0/temp-portfolio-website.git
```

Install dependencies

```bash
  npm install
```

Run the development server

```bash
  npm start
```

Open [localhost:3000](http://localhost:3000) to view in the browser.
